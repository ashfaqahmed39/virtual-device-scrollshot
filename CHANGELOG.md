## [0.1.0] - 2026-07-15

### Added

- Android emulator and USB-device support.
- iOS simulator scroll-page capture through Appium XCUITest.
- Accessibility-anchor stitching with pixel-overlap fallback for iOS.
- Platform-aware target listing and CLI capture selection.
- CLI and JavaScript API.
- Automatic pinned UiAutomator2 setup.
- Appium scroll-to-top and end-of-content detection.
- Pixel-overlap screenshot stitching with system-bar handling.

### Changed

- Renamed the package and CLI to `virtual-device-scrollshot`.
